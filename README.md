# 黒毛和牛 焼肉 KAYA 吉祥寺 Web Site

KAYAの実店舗情報と店舗の特徴を反映した、Cloudflare Pages向け静的サイトです。

## Cloudflare Pages

- Framework preset: `None`
- Build command: 空欄
- Build output directory: `/`

GitHubリポジトリをCloudflare Pagesへ接続すると、push後に自動デプロイされます。

## 公開前の確認

1. `robots.txt` と `sitemap.xml` の `example.pages.dev` を公開ドメインへ変更
2. メニュー・料金・営業時間が最新情報と一致するか店舗側で最終確認
3. 正式に利用可能な店舗写真が届いた場合、`assets/` の画像を差し替え

サイト内の写真は公式サイトの写真を直接コピーせず、店舗の石調壁面・深い赤のシート・黒毛和牛という特徴を踏まえて制作したオリジナル素材です。
